# 2026-ERA-Regression-Progression-Prediction

# Overview

This project builds a predictive model in R to estimate MLB pitcher ERA using historical data (2018–2024). The goal is to identify regression candidates and project future performance based on underlying metrics rather than surface-level results.

# Objectives

- Predict future ERA using advanced pitching metrics

- Identify pitchers likely to improve or regress

- Separate skill from luck-driven performance

# Data

Historical MLB pitching data (2018–2025)

# Key variables include:

- FIP

- xERA

- K% (strikeout rate)

- BB% (walk rate)

- Batted ball metrics

# Methods

- Data cleaning and feature engineering

- Regression modeling (linear / regularized)

- Model evaluation using RMSE


# Key Insight

- ERA alone is not a reliable predictor of future performance. Advanced metrics provide stronger signals for projecting pitcher outcomes.
